# The Run Streak

A personal running blog tracking Garmin runs at [therunstreak.run](https://therunstreak.run).

Black-and-white minimalist Astro 5 static site. No database. Content stored as Markdown files and JSON.

## Phases

1. Static site scaffold (current)
2. Sample content
3. Garmin importer script
4. Automated stats generation
5. Route maps via Leaflet
6. CMS / admin interface
7. Deploy to GitHub Pages via GitHub Actions

## Local dev

```bash
npm run dev
```

## Build

```bash
npm run build
```
